# **App Name**: PulseFolio

## Core Features:

- Centralized Profile Hub: A single landing page aggregating all social media, project, and contact links in a clean, visual grid.
- Smart Bio Assistant: A generative AI tool that analyzes your goals and writes multiple variations of high-impact professional bios.
- Live Customization Engine: Real-time UI controls to adjust color accents, card styles, and layout patterns without technical knowledge.
- Interactive Link Preview: Rich media embedding for links from platforms like YouTube, Spotify, or GitHub directly on the profile.
- Visitor Insights Dashboard: Track unique visits and link click-through rates with an intuitive minimalist interface.

## Style Guidelines:

- Primary color: Electric Blue (#4C82FB), chosen for its sense of professional energy and modern reliability.
- Background color: Deep Charcoal (#0F1218), a highly desaturated version of the primary blue used for a sophisticated dark theme.
- Accent color: Neon Cyan (#68DAFD), an analogous hue used for calls to action and highlighting active states.
- Headline font: 'Space Grotesk' for a tech-forward and architectural look. Body font: 'Inter' for maximum readability in descriptive text.
- Minimalist, thin-stroke line icons that emphasize white space and geometric clarity.
- A vertically centered, single-column layout for mobile-first accessibility, expanding into a responsive grid on desktop.
- Smooth, physics-based stagger animations for card entries and soft glow transitions for hover effects.
